# 🛍️ Customer Segmentation using K-Means Clustering

## 📌 About Dataset

### Context
This dataset is created for learning purposes to understand **customer segmentation**, also known as **market basket analysis**.  
The project demonstrates how to apply **unsupervised machine learning**, specifically the **K-Means Clustering algorithm**, in a simple and practical way.

---

## 📂 Dataset Description

You own a supermarket mall and have collected customer data through membership cards. The dataset includes:

- Customer ID
- Age
- Gender
- Annual Income
- Spending Score

### 💡 Spending Score
Spending Score is a metric assigned to customers based on:
- Purchasing behavior
- Buying patterns
- Customer engagement

---

## 🎯 Problem Statement

As a mall owner, the goal is to:

- Understand customer segments
- Identify **target customers** who are easier to engage
- Help the marketing team develop effective strategies

---

## 🚀 Objectives

By completing this project, you will learn:

1. How to perform customer segmentation using **K-Means Clustering in Python**
2. How to identify target customers for marketing campaigns
3. How segmentation is used in real-world marketing strategies

---

## 🛠️ Technologies Used

- Python 🐍
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

---

## 📊 Methodology

1. Data Preprocessing
2. Feature Selection (Income & Spending Score)
3. Finding optimal clusters using Elbow Method
4. Applying K-Means Clustering
5. Visualizing customer segments

---

## 📈 Expected Outcome

- Clear customer segments based on spending behavior
- Identification of high-value target customers
- Actionable insights for marketing strategies
